## Основные возможности

- **Каталог услуг** с описанием, ценой, длительностью и списком специалистов.
- **Пошаговая онлайн-запись**: выбор услуги → специалист → дата и время → подтверждение.
- **Автоматическая проверка занятости слотов** и защита от двойного бронирования (транзакции с уровнем изоляции Serializable).
- **Личный кабинет клиента**: просмотр, отмена и перенос записей, привязка Telegram-аккаунта.
- **Административная панель**: управление услугами, специалистами, расписанием, записями и базой знаний чат-бота.
- **Telegram-бот** с поддержкой:
  - справочной информации (`/services`, `/price`, `/address`, `/hours`);
  - поиска по базе знаний (FAQ);
  - пошаговой онлайн-записи (для пользователей с привязанным аккаунтом).
- **Разграничение доступа** по ролям: клиент, мастер, администратор.

## Архитектура и стек технологий

Трёхуровневая архитектура на основе шаблона MVC (Model‑View‑Controller).  
Приложение реализовано как монолит — веб‑сервер, сервис чат‑бота и база данных работают в едином окружении.

| Слой | Технология |
|------|------------|
| Серверная платформа | ASP.NET Core 8 (C#) |
| База данных | PostgreSQL 15 |
| ORM | Entity Framework Core 8 |
| Клиентский интерфейс | Razor Views, Bootstrap 5, JavaScript |
| Чат-бот | Telegram.Bot, Long Polling, IHostedService |
| Аутентификация | Сессионная, bcrypt |
| Тестирование | xUnit (модульное), k6 (нагрузочное), OWASP ZAP (безопасность) |
| Развёртывание | Docker Compose |

## Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Незарегистрированный пользователь** | Просмотр каталога услуг и специалистов, справочная информация через веб-чат и Telegram-бота |
| **Клиент (авторизованный)** | Онлайн-запись, отмена и перенос записей, личный кабинет, привязка Telegram |
| **Мастер** | Просмотр своего расписания, отметка о выполнении услуги |
| **Администратор** | Управление услугами, специалистами, расписанием, записями, базой знаний чат-бота, учётными записями |
